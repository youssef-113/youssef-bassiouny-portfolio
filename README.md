# Youssef Bassiouny Portfolio

Welcome to my personal portfolio repository! This project is a modern, responsive, and interactive web application built to showcase my professional background, projects, certifications, and technical skills. It serves as a comprehensive digital resume and a demonstration of my frontend development capabilities.

## About The Project

This portfolio is designed with a focus on user experience, aesthetics, and performance. It features a clean, dark-themed design (with light mode support), smooth animations, and a seamless navigation experience.

**Key Features:**

* **Dynamic Project Showcase:** innovative display of my work with details modals.
* **Interactive UI:** Powered by GSAP animations and responsive design elements.
* **Responsive:** optimized for all devices, from desktops to mobile phones.
* **Certifications & Skills:** A dedicated section to highlight my achievements and technical stack.

## Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

* **Node.js** (v16 or higher recommended)
* **npm** or **yarn**

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/your-username/youssef-bassiouny-portfolio.git
    cd youssef-bassiouny-portfolio
    ```

2. **Install dependencies**

    ```bash
    npm install
    # or
    yarn install
    ```

3. **Start the development server**

    ```bash
    npm run dev
    # or
    yarn dev
    ```

4. **Open your browser**
    Navigate to `http://localhost:5173` (or the URL shown in your terminal) to view the app.
